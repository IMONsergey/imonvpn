# Выбор VPS для imonvpn

Дата: 2026-06-10

## Что нам важно

Для этого проекта VPS должен быть:

- **дешёвым**
- **KVM**
- **x86_64 / amd64**
- с **публичным IPv4**
- на **Debian 12/13** или **Ubuntu 22.04/24.04**
- желательно в **Европе**

## Текущая рекомендация

### 1. Основной выбор: AlphaVPS

Почему:

- официальный EU-провайдер
- есть EU-локации, включая Болгарию и Германию
- есть дешёвые KVM VPS
- цена на момент проверки: от **€2.99/мес** за план `C2G` при годовой оплате

Ссылка:

- [AlphaVPS Cheap VPS](https://alphavps.com/cheap-vps)

## 2. Самый дешёвый альтернативный выбор: RackNerd

Почему:

- официальный сайт показывает KVM VPS от **$2.24/мес**
- заявлены локации `Amsterdam`, `France`, `Dublin`

Риски:

- цена “от” может зависеть от доступности конкретной локации
- EU-локация по минимальной цене может быть недоступна в момент покупки
- в нашем сценарии это вариант “максимально дёшево”, а не “максимально спокойно”

Ссылка:

- [RackNerd](https://www.racknerd.com/)

## 3. Почему не Hetzner как первый выбор

Hetzner технически хороший провайдер, но для этого проекта не первый выбор:

- после изменения цен с **1 апреля 2026** он уже не самый дешёвый
- в нашем проектном контексте считаем риск блокировок для российских сетей выше, чем хотелось бы для стартовой схемы

Ссылки:

- [Hetzner Price Adjustment](https://docs.hetzner.com/general/infrastructure-and-availability/price-adjustment/)
- [Hetzner Cloud](https://www.hetzner.com/cloud)

## Практическое решение

Если нужна одна короткая инструкция без метаний:

1. Сначала смотрим **AlphaVPS**.
2. Берём **самый дешёвый KVM VPS с IPv4** в EU-локации.
3. ОС: **Debian 12**.
4. Если AlphaVPS по какой-то причине не подходит, вторым номером проверяем **RackNerd** с EU-локацией.
