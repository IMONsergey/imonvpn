# Android TV Config Files

В этой папке лежат готовые файлы для импорта в VPN-клиент на Android TV.

## С чего начинать

1. Сначала попробуйте [android-tv-main-primary.txt](./android-tv-main-primary.txt)
2. Если на этой сети будет нестабильно, переключитесь на [android-tv-main-backup.txt](./android-tv-main-backup.txt)

## Дополнительные файлы

- `guest01` и `guest02` — отдельные независимые ключи
- у каждого есть `primary` и `backup`

## Как импортировать

Обычно в приложении на TV:

1. `Import from file`
2. выбрать один из `.txt` файлов из этой папки
3. подключиться

Если приложение не понимает `.txt`, откройте файл, скопируйте строку `vless://...` и импортируйте как текст.
